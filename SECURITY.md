# Security Policy

## Reporting

If you find a security issue in UsageMeter, report it privately. Do not open a public issue.

Include:

- Summary
- Reproduction steps
- Impact
- Affected version or commit

## Scope

In scope:

- UsageMeter desktop app
- Native provider code
- Local cache handling

Out of scope:

- Third-party service outages
- Issues that require a compromised local machine
- Provider account policy decisions
